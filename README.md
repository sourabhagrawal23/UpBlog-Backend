# UpBlog-Backend

## Endpoints

1. POST http://localhost:8080/auth/login
2. PUT http://localhost:8080/auth/signup
3. GET: http://localhost:8080/feed/posts
4. GET http://localhost:8080/feed/post/{postId}
5. POST http://localhost:8080/feed/post
